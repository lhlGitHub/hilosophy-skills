# 海德格尔

## Core Question

技术如何改变人理解世界、理解他人和理解自己的方式？

## Core Insight

技术不只是工具，它会塑造人的观看方式；当一切都被看成可计算、可优化、可调用的资源时，人可能失去对意义的感受。

## Reasoning Pattern

海德格尔的思考方式可以转化为“技术座架模型”：

1. 观察技术把什么变得更可计算、更可控制。
2. 判断这种便利是否改变了人的注意力和价值感。
3. 识别哪些东西被简化成指标、资源或效率对象。
4. 追问人在技术系统中是否仍保留主动理解和意义创造。
5. 为不可量化的经验、关系和责任保留空间。

## Modern Application

适用于：

- 商业：避免把用户只看成数据点和转化对象
- 创业：设计技术产品时保留人的判断与尊严
- 人生：防止效率工具吞噬真实生活
- 技术：分析 AI、自动化、推荐系统和数据化社会

## Example

问题：AI 会不会取代人类？

海德格尔的模型不会只问 AI 能做什么，而会问：

- AI 是否让人把自己也理解成可替换的生产单元？
- 哪些判断被简化成了指标优化？
- 技术是否帮助人更好地理解世界，还是让世界只剩下可调用资源？

真正的问题不是 AI 是否强大，而是人是否仍能在技术系统中保留意义、责任和自由。
