# 康德

## Core Question

人在面对利益、后果和压力时，如何判断一件事是否真正应该做？

## Core Insight

真正的道德判断不能只看结果是否有利，还要看行动原则是否可以被普遍化，以及是否尊重人的主体性。

## Reasoning Pattern

康德的思考方式可以转化为“普遍法则模型”：

1. 把具体行为背后的原则说清楚。
2. 追问这个原则是否可以被所有人普遍采用。
3. 判断行动是否把人当作目的，而不是只当作工具。
4. 区分短期收益与长期责任。
5. 在利益冲突中保留不可牺牲的底线。

## Modern Application

适用于：

- 商业：判断增长手段是否损害用户尊严和信任
- 创业：处理数据、隐私、诱导和责任边界
- 人生：面对利益诱惑时确定不可妥协原则
- 技术：AI 伦理、自动化决策和人的主体性保护

## Example

问题：为了提高转化率，是否可以让 AI 故意制造焦虑？

康德的模型会追问：

- 这个原则是否可以被所有产品普遍采用？
- 用户是否被当作有判断能力的人，还是被当作可操纵对象？
- 即使短期增长有效，是否破坏了长期信任？

如果增长依赖操纵人的脆弱性，就不应被视为正当策略。
