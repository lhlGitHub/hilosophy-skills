# 王阳明

## Core Question

为什么人明明知道道理，却无法行动？真正的理解如何转化为现实改变？

## Core Insight

知不是停留在头脑中的信息，真正的知必然表现为行；行动是理解的完成。

## Reasoning Pattern

王阳明的思考方式可以转化为“知行合一模型”：

1. 检查用户所谓的“知道”是否只是概念认同。
2. 找到阻碍行动的真实欲望、恐惧或代价。
3. 将抽象目标缩小为当下可执行的一步。
4. 用行动反馈修正理解。
5. 在持续行动中形成稳定判断，而不是等待完美确定性。

## Modern Application

适用于：

- 商业：战略必须通过市场行动验证
- 创业：不要沉迷规划，用最小行动测试假设
- 人生：职业选择不能只靠想象，要通过实践获得答案
- 技术：学习新技术必须通过构建真实项目完成理解

## Example

问题：如何提升执行力？

王阳明的模型会指出：

- 如果你真的知道这件事重要，就会在行动中体现出来。
- 没有行动，说明你还没有真正知道，或你回避了某个代价。
- 不要继续寻找更多方法论，先定义今天最小的行动。

行动建议可以是：

- 今天完成一个 25 分钟任务
- 明天复盘阻碍
- 七天内用行动结果检验目标是否真实
