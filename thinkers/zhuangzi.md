# 庄子

## Core Question

人在被功名、成败、评价和单一标准束缚时，如何重新获得精神自由和判断弹性？

## Core Insight

很多痛苦不是来自现实本身，而是来自把局部评价体系误认为绝对世界。

## Reasoning Pattern

庄子的思考方式可以转化为“视角松绑模型”：

1. 识别用户当前被哪一种评价体系绑住。
2. 追问这个评价体系是否真的是唯一标准。
3. 从更大的时间尺度和生命尺度重新观看问题。
4. 区分真实需求与被社会比较制造出来的欲望。
5. 让行动回到更自然、更适合自身生命形态的位置。

## Modern Application

适用于：

- 商业：避免被行业共识和排名焦虑裹挟
- 创业：识别是否只是在追逐风口身份
- 人生：职业选择、焦虑、比较、自我价值
- 技术：避免把效率、增长、可量化指标视为唯一价值

## Example

问题：我是否应该为了更高薪水去做不喜欢的工作？

庄子的模型会提醒：

- “高薪”是哪套评价体系中的成功？
- 这个成功是否真的对应你的生命状态？
- 你是在选择生活，还是在选择别人承认的身份？
- 有没有一种路径让你既不逃避现实，也不出卖自己的节律？

庄子的答案不一定是放弃高薪，而是要求 AI 帮用户从单一标准中退一步，再做选择。
