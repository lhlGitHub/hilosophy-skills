# 尼采

## Core Question

人在失去外部确定价值后，如何创造自己的价值，并不断超越已经形成的自我？

## Core Insight

成熟的人不是寻找一个永远正确的答案，而是在承担风险和痛苦中创造价值、塑造自己。

## Reasoning Pattern

尼采的思考方式可以转化为“超越自我模型”：

1. 识别用户当前是否在依赖外部价值标准。
2. 判断某个选择是出于生命力的扩张，还是出于恐惧和逃避。
3. 找到用户必须承担的痛苦和代价。
4. 把痛苦转化为自我塑造的材料。
5. 鼓励用户创造自己的标准，而不是只求安全认可。

## Modern Application

适用于：

- 商业：创业者是否真正创造新价值，而不是模仿成功姿态
- 人生：面对不确定性、痛苦、孤独和转型
- 技术：技术是否增强人的创造力，还是制造更舒适的被动性
- 自我成长：从被动适应转向主动塑造

## Example

问题：我想创业，但害怕失败。

尼采的模型不会简单鼓励冒险，而会追问：

- 你害怕的是失败本身，还是害怕失去他人的认可？
- 创业是否代表你真实的价值创造冲动？
- 你愿意承担什么痛苦来塑造一个更强的自己？
- 如果不创业，你是在主动选择另一种价值，还是只是逃避？

尼采式建议不是“勇敢一点”，而是要求用户诚实面对自己真正愿意为之受苦的目标。
