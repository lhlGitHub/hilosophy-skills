# 老子

## Core Question

如何在复杂系统中减少强行控制，顺应事物自身的趋势，并以更小代价达成更大效果？

## Core Insight

真正高明的行动不是蛮力推进，而是识别“势”，减少逆势消耗，找到可以借力的位置。

## Reasoning Pattern

老子的思考方式可以转化为“顺势模型”：

1. 先观察系统的自然流向，而不是急于表达主观愿望。
2. 区分可改变的局部与不可改变的大势。
3. 减少过度干预，避免把资源消耗在对抗趋势上。
4. 寻找杠杆点，让小行动触发系统本身的力量。
5. 保持柔性，根据反馈调整，而不是用固定计划压制变化。

## Modern Application

适用于：

- 商业：判断市场方向和进入时机
- 创业：选择顺应趋势的切入点
- 人生：减少和自身条件、时代环境的硬碰硬
- 技术：识别技术扩散中的自然路径

## Example

问题：现在要不要进入 AI 创业？

老子的模型不会先问“我想不想创业”，而是先问：

- AI 是否正在成为基础设施？
- 用户需求是否已经自然涌现？
- 竞争中是否存在被巨头忽略的小切口？
- 我的资源能否借到平台、模型、渠道或社区的势？

如果趋势强、切口小、行动轻、反馈快，则可以进入。  
如果只是被热度推动，既无用户痛点，也无资源杠杆，就是逆势中的盲动。
