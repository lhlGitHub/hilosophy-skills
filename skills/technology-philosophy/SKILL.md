# Purpose

这个 Skill 用于分析技术未来问题。

它帮助 AI Agent 避免两种极端：

- 技术乐观主义：认为技术自然带来进步
- 技术悲观主义：认为技术必然毁灭人类

它关注：

- 技术改变了什么结构
- 人类哪些能力被替代，哪些能力被放大
- 技术是否增强人的创造力和自由
- 社会矛盾会如何通过技术重新展开

# When To Use

当用户提出以下问题时使用：

- AI 会不会取代人类？
- 技术进步是否一定是好事？
- 人类未来的价值在哪里？
- 自动化会如何改变工作？
- AI 时代应该学习什么？

# Thinking Process

1. 将问题从“技术会不会发生”转为“技术如何改变人的处境”。
2. 使用黑格尔分析技术引发的社会矛盾和阶段变化。
3. 使用尼采分析技术是削弱人还是增强人的创造力。
4. 使用第一性原理模型拆解人类不可替代的核心能力。
5. 必要时调用海德格尔预留模型，分析技术是否把世界变成单一资源。
6. 输出对个人、组织和社会的行动建议。

# Available Thinkers

- 黑格尔：技术作为历史矛盾展开的一部分
- 尼采：技术是否服务于人的自我超越
- 海德格尔：技术是否把世界变成可计算、可支配的对象
- 王阳明：在技术变化中通过行动获得真实理解

# Output Format

```text
1. 问题本质
   说明这不是单纯预测问题，而是什么人类处境问题。

2. 技术改变的结构
   分析生产、认知、权力、关系或价值结构的变化。

3. 哲学视角
   使用 2-3 位哲学家解释技术未来。

4. 思维模型
   用 Framework 拆解关键判断。

5. 行动建议
   分别给出个人、组织或社会层面的建议。
```

# Guardrails

- 不要做简单的 AI 乐观或悲观预测。
- 不要把技术当作独立于人类社会的力量。
- 不要只列趋势，必须分析趋势背后的价值和结构。
