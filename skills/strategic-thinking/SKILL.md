# Purpose

这个 Skill 用于处理战略问题。

它帮助 AI Agent 判断：

- 一个方向是否顺应时代趋势
- 当前竞争是否处在结构性变化中
- 用户应该等待、进入、转向还是退出
- 哪些行动可以借力，哪些行动是在逆势消耗

# When To Use

当用户提出以下问题时使用：

- 我要不要进入某个行业？
- 现在是否适合创业？
- 公司应该如何定位？
- 如何判断一个趋势是真的还是假的？
- 面对强大竞争者应该怎么办？
- 长期战略应该如何选择？

# Thinking Process

1. 判断问题所处的历史阶段和外部环境。
2. 识别趋势、阻力、矛盾和用户自身资源。
3. 使用老子的顺势模型判断势能方向。
4. 使用黑格尔的辩证发展模型判断结构性矛盾。
5. 使用王阳明的知行合一模型判断是否能通过行动验证。
6. 将哲学判断转化为现实战略选项。
7. 输出可执行建议，而不是抽象概念。

# Available Thinkers

- 老子：顺势、无为、借力、减少逆势行为
- 黑格尔：矛盾、阶段、历史发展、否定之否定
- 王阳明：知行合一、行动验证、良知决断
- 尼采：价值创造、意志强度、是否主动塑造未来

# Output Format

```text
1. 问题本质
   用一句话说明这不是表面问题，而是什么战略问题。

2. 哲学视角
   分别说明调用哪些哲学家，以及他们如何看这个问题。

3. 思维模型
   使用 1-3 个 Framework 展开推理步骤。

4. 现实应用
   将推理落到行业、资源、时机、竞争和风险。

5. 行动建议
   给出短期行动、中期验证、长期判断标准。
```

# Guardrails

- 不要输出哲学家生平。
- 不要把哲学概念当作结论。
- 不要只说“顺势而为”，必须指出具体的势。
- 不要把战略建议伪装成确定性预测。
