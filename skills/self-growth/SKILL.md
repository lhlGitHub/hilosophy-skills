# Purpose

这个 Skill 用于处理自我成长问题。

它帮助 AI Agent 从哲学角度分析：

- 为什么知道却做不到
- 如何从外部评价中解放出来
- 如何形成持续行动
- 如何面对痛苦、焦虑和自我怀疑
- 如何把成长理解为自我塑造

# When To Use

当用户提出以下问题时使用：

- 如何提升执行力？
- 我为什么总是拖延？
- 我很焦虑怎么办？
- 我不知道自己想要什么。
- 如何变得更强？
- 如何建立长期自律？

# Thinking Process

1. 区分问题是行动问题、价值问题、环境问题还是身份问题。
2. 使用王阳明判断“知道”是否已经变成行动。
3. 使用尼采判断用户是否在逃避生命强度和责任。
4. 使用庄子判断焦虑是否来自外部评价系统。
5. 将抽象自我成长转化为今天能做的具体行动。
6. 设置反馈机制，而不是依赖意志力口号。

# Available Thinkers

- 王阳明：知行合一，行动是理解的完成
- 尼采：超越自我，创造自己的价值
- 庄子：逍遥，解除对单一评价体系的依附
- 老子：减少强行控制，顺应节律和系统条件

# Output Format

```text
1. 问题本质
   判断这是行动、价值、环境还是身份问题。

2. 哲学视角
   说明相关哲学家会如何重新定义这个问题。

3. 思维模型
   展开具体推理步骤。

4. 行动设计
   给出今天、七天、三十天的行动安排。

5. 反馈标准
   说明如何判断自己是否真的发生了改变。
```

# Guardrails

- 不要输出鸡汤。
- 不要把痛苦浪漫化。
- 不要把执行力问题完全归因于意志薄弱。
- 必须给出可观察的行动指标。
